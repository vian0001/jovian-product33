TOPOLOGI JARINGAN WIRELESS

Tampilan topologi di gns3

Topologi jaringan wireless menunjukkan bagaimana perangkat MikroTik terhubung dengan klien nirkabel. Konfigurasi security profile dan enkripsi WPA2 merupakan komponen penting untuk mengamankan jaringan nirkabel dari akses tidak sah.

Router MikroTik akan berfungsi sebagai Access Point (AP) yang menyediakan koneksi nirkabel terenkripsi bagi perangkat klien. Security profile menentukan parameter keamanan seperti metode autentikasi dan jenis enkripsi yang digunakan pada jaringan wireless.

Berikut adalah komponen-komponen utama dalam topologi ini:

1. MikroTik Router
# Berfungsi sebagai wireless access point dengan security profile WPA2.
2. Klien Wireless (Laptop, Smartphone, IoT devices)
# Perangkat yang terhubung ke jaringan nirkabel melalui autentikasi WPA2.
3. SSID (Service Set Identifier)
# Nama jaringan nirkabel yang terlihat oleh perangkat klien.

Konfigurasi security profile yang tepat sangat penting untuk mencegah serangan terhadap jaringan wireless. Menggunakan standar WPA2 dengan passphrase yang kuat adalah praktik keamanan dasar untuk melindungi jaringan Anda dari akses tidak sah.

1. SECURITY PROFILE DAN STANDAR WPA2

Pengertian Security Profile pada MikroTik

Dari sumber MikroTik Documentation (2024), Security Profile pada RouterOS adalah serangkaian konfigurasi keamanan yang diterapkan pada interface wireless untuk mengontrol bagaimana perangkat klien dapat terhubung dan mengakses jaringan. Security profile mencakup berbagai parameter seperti metode autentikasi, jenis enkripsi, dan kunci keamanan yang digunakan pada jaringan wireless.

Dari sumber NetworkLessons (2023), Security Profile pada MikroTik berfungsi sebagai templat keamanan yang dapat diterapkan ke satu atau beberapa interface wireless. Ini memungkinkan administrator untuk mengelola keamanan jaringan nirkabel secara terpusat dan konsisten, terutama pada perangkat MikroTik dengan multiple wireless interface.

Standar Enkripsi WPA2

Dari sumber TechTarget (2024), WPA2 (Wi-Fi Protected Access 2) adalah standar keamanan wireless yang dikembangkan oleh Wi-Fi Alliance sebagai penerus WPA. WPA2 menggunakan algoritma enkripsi AES (Advanced Encryption Standard) yang jauh lebih kuat dibandingkan dengan TKIP yang digunakan pada WPA generasi pertama. Standar WPA2 resmi dirilis pada tahun 2004 dan menjadi mandatory pada semua perangkat bersertifikasi Wi-Fi sejak 2006.

Dari sumber CBT Nuggets (2023), WPA2 mendukung dua mode utama: WPA2-Personal (WPA2-PSK) yang menggunakan pre-shared key dan cocok untuk jaringan rumah atau usaha kecil, serta WPA2-Enterprise yang menggunakan server RADIUS untuk autentikasi dan lebih cocok untuk lingkungan bisnis skala besar. WPA2 dengan AES-CCMP menawarkan keamanan yang sangat kuat dan telah menjadi standar industri selama bertahun-tahun sebelum kedatangan WPA3.

Dari 2 pendapat tersebut dapat disimpulkan bahwa Security Profile pada MikroTik adalah komponen konfigurasi keamanan yang penting untuk mengamankan jaringan wireless, sedangkan WPA2 adalah standar keamanan yang menggunakan enkripsi AES yang kuat untuk melindungi data pada jaringan nirkabel, tersedia dalam mode Personal (PSK) untuk implementasi sederhana dan Enterprise untuk lingkungan bisnis yang memerlukan keamanan tingkat tinggi.

2. MENGKONFIGURASI SECURITY PROFILE WPA2

Tampilan topologi di gns3

Membuat Security Profile WPA2 melalui Winbox (Step-by-step)

Berikut adalah panduan langkah demi langkah menggunakan Winbox untuk membuat Security Profile WPA2 ‚Äî tidak menggunakan CLI. Ikuti setiap langkah pada antarmuka Winbox.

# Langkah-langkah melalui Winbox untuk membuat security profile WPA2
1. Buka Winbox dan hubungkan ke router MikroTik (masukkan MAC atau IP, klik Connect).
2. Pada jendela Winbox, pilih menu Wireless di sidebar.
3. Klik tab Security Profiles.
4. Klik tombol + (Add) untuk menambahkan profile baru.
5. Pada kolom Name, isi: profile-wpa2 (atau nama yang diinginkan).
6. Pada Mode, pilih dynamic-keys.
7. Pada Authentication Types, centang WPA2 PSK.
8. Masukkan password pada kolom WPA2 Pre-Shared Key (contoh: StrongPassw0rd!) ‚Äî ganti dengan passphrase yang kuat.
9. Pada bagian cipher, atur Unicast Ciphers = aes-ccm dan Group Ciphers = aes-ccm.
10. Atur Group Key Update jika perlu (mis. 5m), dan Management Protection = allowed atau required.
11. Klik OK untuk menyimpan security profile baru.

Catatan: selalu gunakan passphrase minimal 12 karakter dengan kombinasi huruf besar, huruf kecil, angka, dan simbol untuk mencegah serangan brute force.

Melihat Detail Security Profile (Winbox)

Untuk melihat detail profile yang telah dibuat:

# Cara melihat detail Security Profile via Winbox
1. Winbox -> Wireless -> Security Profiles tab.
2. Pilih profile yang ingin dilihat, lalu klik Inspect atau Double-click untuk membuka detail.
3. Periksa pengaturan seperti authentication-types, unicast-ciphers, group-ciphers, dan group-key-update.

Pembuatan security profile WPA2 di Winbox menawarkan antarmuka grafis yang mudah diikuti ‚Äî semua parameter penting (mode, authentication types, cipher, pre-shared-key, management protection) dapat diatur tanpa perlu mengetik perintah CLI.

3. MENERAPKAN SECURITY PROFILE KE INTERFACE WIRELESS

Tampilan topologi di gns3

Konfigurasi Interface Wireless melalui Winbox (Step-by-step)

Langkah-langkah berikut menunjukkan bagaimana menerapkan security profile WPA2 yang telah dibuat ke interface wireless (mis. wlan1) menggunakan Winbox.

# Langkah-langkah menerapkan security profile ke interface WLAN (Winbox)
1. Winbox -> Wireless -> tab Interfaces.
2. Cari interface wireless yang akan dikonfigurasi (mis. wlan1), lalu double-click pada baris interface tersebut.
3. Pada jendela properti interface, buka tab Wireless.
4. Atur parameter: Mode = ap-bridge, SSID = MikroTik-Secure, Band = 2ghz-b/g/n, Channel Width = 20/40mhz-Ce, Frequency = auto.
5. Pada dropdown Security Profile, pilih profile WPA2 yang sudah dibuat (mis. profile-wpa2).
6. Centang atau pastikan Enabled agar interface aktif.
7. Klik OK untuk menyimpan konfigurasi interface.

Verifikasi Penerapan (Winbox)

Berikut cara memeriksa bahwa security profile telah diterapkan dengan benar:

# Cara verifikasi penerapan via Winbox
1. Winbox -> Wireless -> Interfaces: pastikan kolom security-profile pada baris wlan1 menunjukkan nama profile yang dipilih dan status Running (ditandai R).
2. Winbox -> Wireless -> Registration (Registration Table): lihat daftar klien yang terhubung (MAC, signal, tx-rate).
3. Winbox -> Logs (gunakan filter atau cari topik berkaitan wireless) untuk melihat peristiwa autentikasi dan error.

Penerapan security profile ke interface wireless melalui Winbox mudah dan visual ‚Äî Anda dapat langsung melihat dan mengubah parameter interface tanpa harus menggunakan perintah CLI.

4. PENGUJIAN DAN PRAKTIK KEAMANAN TERBAIK

Tampilan topologi di gns3

Pengujian Konfigurasi WPA2 (Winbox)

Uji konfigurasi WPA2 menggunakan antarmuka Winbox dengan langkah-langkah berikut:

# Langkah-langkah pengujian via Winbox
1. Winbox -> Wireless -> Security Profiles: pastikan profile terlihat dan konfigurasi sesuai.
2. Winbox -> Wireless -> Interfaces: cek bahwa wlan1 sudah memakai security profile yang benar dan statusnya berjalan.
3. Winbox -> Wireless -> Registration: sambungkan perangkat klien (laptop/smartphone) dan periksa apakah klien muncul di daftar dengan enkripsi aktif.
4. Winbox -> Wireless -> klik tombol Scan pada interface untuk memeriksa interferensi kanal dan melihat SSID lain di area.
5. Winbox -> Logs: cek log untuk peristiwa autentikasi, deauth, atau error terkait wireless.
6. Untuk monitoring trafik singkat: Winbox -> Interfaces -> pilih wlan1 -> klik Traffic (atau gunakan Tools -> Torch untuk analisis lebih lanjut).

Praktik Keamanan Terbaik untuk WPA2 (Winbox)

Berikut praktik terbaik yang dapat langsung diterapkan lewat Winbox:

  1. Gunakan password yang kuat dan panjang (minimal 12 karakter) dengan kombinasi huruf besar, huruf kecil, angka, dan simbol
  2. Aktifkan Management Protection (set ke required jika memungkinkan) untuk mencegah serangan deauthentication
  3. Gunakan Unicast Ciphers dan Group Ciphers = aes-ccm saja; hindari TKIP
  4. Perbarui firmware RouterOS secara berkala untuk mendapatkan perbaikan keamanan terbaru
  5. Atur Group Key Update ke nilai yang tepat (mis. 5 menit) untuk mengurangi risiko key reuse
# Contoh konfigurasi security profile (Winbox) dengan praktik terbaik
1. Winbox -> Wireless -> Security Profiles -> + (Add)
2. Name: best-practice
3. Mode: dynamic-keys
4. Authentication Types: WPA2 PSK
5. Unicast Ciphers: aes-ccm
6. Group Ciphers: aes-ccm
7. WPA2 Pre-Shared Key: V3ryStr0ngP@ssw0rd! (ganti dengan passphrase Anda)
8. Group Key Update: 5m
9. Management Protection: required
10. WPS Mode: disabled
11. Klik OK

Jika perangkat klien dan RouterOS mendukung, pertimbangkan juga menyusun Security Profile untuk mode transisi WPA2/WPA3 melalui Winbox:

# Mode transisi WPA2/WPA3 (Winbox, RouterOS v7+ dengan wifiwave2)
1. Winbox -> Wireless -> Security Profiles -> + (Add)
2. Name: transition-mode
3. Mode: dynamic-keys
4. Authentication Types: centang WPA2 PSK dan WPA3 PSK jika tersedia
5. Unicast Ciphers / Group Ciphers: aes-ccm
6. Masukkan kedua passphrase pada kolom WPA2 / WPA3 Pre-Shared Key (atau gunakan passphrase yang sama jika diperlukan)
7. Management Protection: required, WPS Mode: disabled
8. Klik OK

Untuk RouterOS versi 7 ke atas, konfigurasi wireless pada package wifiwave2 menggunakan path antarmuka yang sedikit berbeda di Winbox ‚Äî namun konsep pengaturan security profile, authentication-types, encryption, dan passphrase tetap sama.

Meskipun WPA2 masih aman jika dikonfigurasi dengan benar, WPA2 memiliki kelemahan seperti kerentanan KRACK (Key Reinstallation Attack). Pertimbangkan beralih ke WPA3 jika perangkat klien Anda mendukungnya, terutama untuk jaringan yang menangani data sensitif.

Pengujian dan penerapan praktik keamanan terbaik melalui Winbox memungkinkan administrator untuk mengelola konfigurasi secara visual dan cepat. Pastikan untuk melakukan verifikasi, monitoring, dan memperbarui firmware secara berkala.
